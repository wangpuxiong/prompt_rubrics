这是一个基于 Java 实现的反井字棋（Anti-Tic-Tac-Toe）游戏项目，旨在提供一个有趣的策略游戏体验。该项目解决了传统井字棋游戏规则单一的问题，通过反转游戏规则增加了游戏的挑战性和趣味性。
游戏采用 5x5 棋盘，核心规则是：如果玩家恰好连成 3 个棋子（横向、纵向或对角线），该玩家就会输掉游戏，对手获胜。这种"连成即输"的反转规则要求玩家在追求胜利的同时必须避免形成三连，增加了策略深度。游戏支持双人对战模式，通过命令行交互界面进行操作，玩家需要输入行列坐标来放置棋子。游戏会实时显示棋盘状态，检测胜负条件，并在棋盘填满时判定平局。
技术栈方面，项目使用 Java 语言开发，基于 Eclipse Temurin OpenJDK 21 JDK，采用面向对象设计，核心逻辑封装在 TicTacToe 类中。项目提供了 Docker 容器化部署方案，使用 Eclipse Temurin 21-jdk 作为基础镜像，方便在不同环境中运行和开发。
