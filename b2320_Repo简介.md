zip-zop-zoop flattener 是一个基于 PyQt5 的桌面应用程序，用于扁平化处理嵌套的 ZIP 压缩文件。该项目旨在解决多层嵌套 ZIP 文件难以直接使用的问题，通过自动解压所有内层 ZIP 文件并重新打包，将复杂的嵌套结构转换为扁平的文件结构。

该工具提供了两种处理模式：普通扁平化模式会解压所有嵌套的 ZIP 文件并删除原始内层压缩包；超级扁平化模式在此基础上还会移除单层顶层文件夹，进一步简化文件结构。应用采用拖放式操作界面，支持批量处理多个文件夹，使用多线程技术确保界面响应流畅，并提供详细的处理日志。当处理过程中出现文件名冲突时，系统会自动添加随机后缀以确保文件完整性。

技术栈方面，项目使用 Python 3.8+ 作为开发语言，PyQt5 构建图形用户界面，利用 Python 标准库 zipfile 模块处理压缩文件操作，并提供了 Docker 容器化部署方案，方便在不同环境中运行。
